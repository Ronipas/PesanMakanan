# 🍔 PesanMakanan App

Aplikasi web sederhana untuk memesan makanan yang dibuat menggunakan **React JS**.  
Project sederhana ini berfokus pada Front End dengan menggunakan API dan JSON. 
Aplikasi ini bisa menghitung total biaya dari pesanan secara realtime.

---

## 🚀 Tech Stack

Aplikasi ini dibangun menggunakan beberapa teknologi berikut:

- ⚛️ React JS (v19)
- 🎨 Bootstrap
- ⭐ Font Awesome
- 🔗 Axios
- 🔔 SweetAlert
- 🧭 React Router
- 🗂 JSON Placeholder (Local API)
- 📄 JSON

---

## ✨ Fitur Aplikasi

Beberapa fitur utama dari aplikasi **PesanMakanan**:

- Menampilkan daftar menu makanan
- Melihat detail makanan
- Menambahkan makanan ke pesanan
- Navigasi antar halaman menggunakan **React Router**
- Notifikasi interaktif menggunakan **SweetAlert**
- Mengambil data dari **API menggunakan Axios**
